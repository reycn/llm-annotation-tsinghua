# Data Annotation with Large Language Models
Tutorial materials on using large language models for large-scale data annotation, including course slides, referenced papers, and experimental code.
![img](./misc/simulation.png)

# Repository Structure
```
- ./experiments/ # Simulation experiment code covered in this tutorial
    - 1.classification.ipynb # sentiment and topic classification
    - 2.imputation.ipynb # data imputation
    - 3.scaling.ipynb # scaling your data process
    - 4.simulation.ipynb # population and sample parameters
- ./papers/ # Papers mentioned in this tutorial
- ./slides/ # Course slides
- ./README.md # [README in Chinese]
```

# Main Topics
- Data annotation using large language models, missing value imputation, and prompt engineering
- Techniques for large-scale processing
- Quality assessment methods
- Limitations of large language models
